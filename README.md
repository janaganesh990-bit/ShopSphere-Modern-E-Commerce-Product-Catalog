# 🛍️ ShopSphere

A modern and responsive e-commerce product catalog built with HTML, CSS, and JavaScript.

## ✨ Features

- Product Catalog
- Product Details
- Live Search
- Category Filter
- Price Sorting
- Shopping Cart
- Local Storage
- SPA Routing
- REST API Integration
- Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Local Storage
